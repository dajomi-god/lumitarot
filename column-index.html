루미타로 칼럼

타로 카드와 뇌과학으로 읽는 연애 심리학.
왜 우리는 이렇게 사랑하고, 이렇게 아파할까요.

#1
자이가르닉 효과

이별 후 왜 자꾸 생각날까

끝난 관계인데 머릿속을 떠나지 않는 사람. 의지가 약해서가 아니에요. 뇌가 미완결된 일을 처리하려는 본능 때문이에요.

→
#2
도파민 보상 시스템

도파민 중독과 썸의 심리학 — 컵 7이 말하는 것

썸이 실제 연애보다 더 설레는 이유. 뇌가 불확실성을 먹고 자라는 도파민 괴물이기 때문이에요.

→
#3
회피형 애착

회피형 연애, 뇌과학으로 이해하기 — 은둔자 카드

좋아하면서 밀어내는 사람. 나쁜 사람이 아니에요. 뇌가 친밀함을 위협으로 인식하도록 학습된 거예요.

→
#4
사회적 배제 · 통증 회로

연락이 끊겼을 때 뇌에서 일어나는 일

잠수, 읽씹, 갑자기 차가워진 온도. 연락이 끊기는 순간 뇌는 실제 신체 통증과 동일한 반응을 보내요.

→
#5
간헐적 강화

왜 나쁜 사람한테 더 끌릴까

항상 잘해주는 사람보다 잘해주다 차갑게 구는 사람한테 더 끌리는 이유. 당신 취향이 이상한 게 아니에요.

→
#6
애착 이론

애착 유형별 연애 패턴 — 나는 어떤 유형일까

불안형, 회피형, 안정형. 내가 연애에서 반복하는 패턴은 어린 시절부터 뇌에 새겨진 애착 회로예요.

→
#7
편도체 방어 기제

재회를 방해하는 뇌의 방어 기제

진심으로 변했는데 상대는 왜 믿지 않을까. 이별 후 뇌에 자동으로 생성되는 방어 회로를 분석해요.

→
#8
옥시토신 · 도파민

옥시토신과 장기연애 — 설렘이 사라진 이유

사귄 지 1~2년이 지나면 왜 설렘이 줄어들까. 감정이 식은 게 아니라 뇌의 신경전달물질이 바뀐 거예요.

→
#9
손실 회피 편향

손실 회피 편향 — 이별이 득보다 아픈 이유

새로운 좋은 사람을 만나도 떠난 사람이 더 크게 느껴지는 이유. 뇌는 손실을 이득보다 2.5배 강하게 느껴요.

→
#10
불안 애착 · 달 카드

달 카드와 불안 애착 — 의심이 많은 사람의 뇌

연애만 하면 유독 불안하고 확인하고 싶어지는 이유. 달 카드가 상징하는 불안 애착을 뇌과학으로 읽어요.

→
#11
거울 뉴런

거울 뉴런 — 상대방 감정을 내가 느끼는 이유

상대가 차갑게 굴면 나도 위축되는 이유. 거울 뉴런이 감정을 그대로 복사하는 메커니즘을 분석해요.

→
#12
집착 · 악마 카드

악마 카드와 집착의 심리학

끊고 싶은데 끊지 못하는 관계. 뇌의 보상 회로가 어떻게 집착을 만드는지 분석해요.

→
#13
세로토닌 자립 · 세계 카드

솔로가 편한 사람의 뇌 — 세계 카드와 자립심

혼자가 더 편하고 연애가 귀찮은 이유. 자립형 뇌의 특징과 나에게 맞는 인연이 오는 방식.

→
#14
전전두엽 과부하 · 소드

소드 카드가 자꾸 나오는 사람의 심리

타로를 볼 때마다 소드가 나오는 이유. 생각 과부하 상태의 뇌와 벗어나는 방법.

→
#15
신경 회로 패턴

타로가 말하는 나의 연애 패턴

왜 매번 비슷한 연애를 반복할까. 타로 카드는 내 뇌의 연애 회로를 거울처럼 보여주는 도구예요.
